# Pulse CRM Dashboard

A clean and modern **Product Analytics Dashboard** for Pulse CRM.

### Features
- Monthly Active Users, Activation Rate, Churn Rate, MRR
- New Signups vs Active Users chart
- Plan Distribution (Pro / Starter / Free)
- Recent Product Events table
